# Origin Replay

Pipeline replay is integrated into **history-fixer** as the **Pipeline Replay** tab.

Run the operator console:

```powershell
.\scripts\run-operator.ps1
```

Open http://127.0.0.1:5174, select a missile event, switch to **Pipeline Replay** (or use **Replay pipeline** in the audit panel).

This standalone folder is kept for reference; use `history-fixer/` for local development.
